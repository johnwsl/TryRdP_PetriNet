2-Modelagem Básica

  1. 2.9 Rede de Petri Estocástica

      As redes de Petri originais não possuem noção de tempo. A introdução da duração dos eventos resultou em um novo tipo de rede de Petri chamada de rede de Petri temporizada. Um caso particular das redes de Petri temporizadas são as redes de Petri estocásticas (do inglês, stochastic Petri nets (SPNs)). Nas SPNs, os atrasos dos eventos (representados como transições) são considerados variáveis aleatórias com uma distribuição exponencial. Em TryRdP, usaremos uma extensão das SPNs chamada de rede de Petri estocástica generalizada (GSPN) que considera dois tipos de transições: as temporizadas e as imediatas. Mas note que o tempo de disparo distribuído exponencialmente está sempre associado às transições temporizadas, uma vez que às transições imediatas, por definição, disparam com tempo zero. Para fins de concisão, o acrônimo SPN em TryRdP será usado para expressar toda a família de modelos derivados das SPNs. Caso deseje se aprofundar no assunto, você poderá ler o artigo link .

      Um outro conceito importante associado às transições temporizadas é o grau de habilitação. Esse conceito indica se uma determinada transição pode fazer disparos simultâneos. O grau de habilitação pode ser de três tipos: Single-Server, Multiple-Server ou Infinite-Server.

  • Single-Server: não é possível realizar disparos simultâneos.
  • Multiple-Server: é possível fazer um número finito "n" de disparos simultâneos.
  • Infinite-Server: é possível fazer infinitos disparos de uma única vez.

      Abaixo é apresentado um exemplo de uma transição temporizada e suas propriedades. Nas propriedades vemos: o "Label" que é o nome dado à transição (esse nome precisa ser único); o "Tempo" que é o atraso (delay) exponencialmente distribuído associado à transição; o "Tipo de Servidor": é o grau de habilitação (no TryRdP usaremos apenas o "single server" (servidor único)); a "Distribuição de Probabilidade": é a distribuição adotada (no TryRdP usaremos apenas a distribuição "exponencial").
